# SetFriStart(DateTime friStart)

The work hour start for Fridays. Note that only the time part of the DateTime is used

## Parameters

* **friStart** DateTime

```crmscript
NSTicketPriorityEntity thing;
DateTime friStart;
thing.SetFriStart(friStart);
```
