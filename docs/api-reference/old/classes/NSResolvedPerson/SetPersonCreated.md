# SetPersonCreated(Bool personCreated)

Indicates if the resolved person was created or not.

## Parameters

* **personCreated** Bool

```crmscript
NSResolvedPerson thing;
Bool personCreated;
thing.SetPersonCreated(personCreated);
```
