# Integer GetTheme()

Predefined themes for the chat widget

**Returns:** Integer

## Enum: theme

* 0 = Classic
* 1 = Modern

## Example

```crmscript
NSChatWidgetSettings thing;
Integer theme  = thing.GetTheme();
```
