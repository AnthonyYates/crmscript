# SetCreatedBy(NSAssociate createdBy)

The person that first created the document. The property is read-only.

## Parameters

* **createdBy** NSAssociate

```crmscript
NSBlobEntity thing;
NSAssociate createdBy;
thing.SetCreatedBy(createdBy);
```
