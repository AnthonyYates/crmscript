# SetIsSystemTask(Bool isSystemTask)

If IsSystemTask is true, the task is not initiated by an associate.

## Parameters

* **isSystemTask** Bool

```crmscript
NSBatchTaskInfo thing;
Bool isSystemTask;
thing.SetIsSystemTask(isSystemTask);
```
