# SetContact(NSContact contact)

The contact associated with the appointment. It may also be null if no contact is associated with the appointment.

## Parameters

* **contact** NSContact

```crmscript
NSAppointmentEntity thing;
NSContact contact;
thing.SetContact(contact);
```
